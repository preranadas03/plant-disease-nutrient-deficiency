# Plant Disease and Nutrient Deficiency Detection

## Model

YOLOv8 classification model for cotton leaf disease detection with nutrient deficiency mapping.

## Classes

- Bacterial Blight
- Curl Virus
- Healthy Leaf
- Herbicide Growth Damage
- Leaf Hopper Jassids
- Leaf Redding
- Leaf Variegation

## Run

```bash
pip install -r requirements.txt
python code/nutdef.py
```
